# VROOM – Car Rental Service (Frontend + Backend)

VROOM is a car rental web application designed to allow users to search, view, and reserve cars easily.
This project includes user authentication, search filters, car listings, and booking flow with a mock payment
timer.

## Features
- Home page with CTA buttons
- Offers page with weekend/weekly discounts
- Car search with filters (car type, capacity, price)
- Real-time car availability
- Reservation flow with 30-second payment timer
- Booking confirmation with generated booking ID
- User login & signup (age and license declaration check)
- Admin-level capacity management (each car model has limited units)
- Fake payment system

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js, MongoDB
- **Version Control:** Git & GitHub

## Project Structure
- `/index.html` – Home page
- `/cars.html` – Car search results + filters
- `/car-details.html` – Individual car detail page
- `/booking.html` – Booking flow
- `/confirmation.html` – Final confirmation with booking ID
- `/auth/` – Login & Signup pages
- `/backend/` – All backend code (API, database, auth, booking logic)
